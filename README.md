# 🎯 Redrob Intelligent Candidate Ranking Engine

Multi-signal hybrid scorer for the Redrob Hackathon 2024.

**No GPU · No API calls · Fully offline · Runs in seconds**

## How to use
1. Paste or edit the Job Description on the left
2. Upload your `candidates.jsonl` (or use the built-in 50-candidate sample)
3. Click **Rank Candidates**
4. Get a shortlist with scores, reasoning, and a downloadable CSV
